Beginner concepts

What is the internet?

The internet is computers sharing information world wide.
Text, photos, videos; music -- most content exists as html files.
Html = hyper text markup language
These html files are stored on local servers around the globe.

A local server in Kathmandu can talk with a server in Boston
using http, or hypertext transfer protocol.
Your request for information using a Google search
sends a retrieval of the correct html files using http.
Then, your computer browser interprets the markup language
in order to display a web page to you.

Markup language, or, Tags and Elements

Tags are what computers need to read in order to display text a certain way.

For example, bolding can be done using at least two different tags
one using a "b" in brackets < > before and after the text,
and the other using the word "strong." In this course we haven't yet
learned Why there is this difference and when to use which
(though my hunch is that the "strong" tag is used in CSS and is more efficient.)

Elements are the content, for example, headings, titles and images.
Elements can be inline or block. Bolding, italicizing, anchor tags (indicating a web site link)
or img (image) tags are used "inline."

Clicking here on hyperlink text is an inline command in the code line.

Tags directing the computer to display a new paragraph or create
a line break are examples of "block" tags.
What does the "form" tag do?
It creates a return carriage to begin a new paragraph after the end tag. Huh.
These block commands also signal an "invisible box" around the content inside that element.
Synonomous terms are span (inline) and div (block).

Computers are stupid, ie, literal.

You have to code your command exactly.
They cannot interpolate a typo like a human would. One missing end tag, and your sentence will bold on forever!
